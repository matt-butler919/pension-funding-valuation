Pension Funding Valuation project

Link to full project - https://docs.google.com/spreadsheets/d/1-eBQuII54UnZMGhYl8yiv6VZEhSa-6ijmXjxQ_HtTdk/edit?gid=1989453085#gid=1989453085

Census data randomly generated, mortality table data from SOA - https://www.soa.org/resources/experience-studies/2014/research-2014-rp/

A pension funding valuation model is used to assess the long term financial health of a retirement plan by comparing accumulated assets against promised future liabilities. In this project, I built a defined benefit pension funding valuation model using a randomly generated dataset of 100 hypothetical participants. I used the SOA RP-2014 mortality rates to derive gender-specific survival rates and annuity factors, then projected each participant's benefit using a 1.5% × service × salary accrual formula with a 3.5% salary scale assumption, while also applying joint and survivor adjustments. Finally, I aggregated results to a summary dashboard reporting total PBO, ABO, funded status, and funding ratio.
